# SketchFoxsky's VRCWorldTools
Collection of useful tools and fun gimmicks for VRChat worlds!

## Uno Card Game Prefab
Read **setup** and **documentation** for the prefab [(HERE)](https://github.com/SketchFoxsky/VRCWorldTools/tree/main/Assets/SketchFoxsky/Uno#setup-and-documentation).

Fully functional card game to drag and drop into your world!

* PCVR and StandaloneVR support!
* Supports up to 8 players by default; with **midgame join** and **late player syncing**!
* **Fully animated cards** for drawing, playing, and pickup/drop from the player hand!
* Optional card privacy to prevent users from seeing or touching your hand!

Requires UdonSharp!
